# Lessons from the multi-agent run

The complete English document is available at
[`../../docs/phylo-codex-learnings.en.md`](../../docs/phylo-codex-learnings.en.md).

Key lessons:

- The eight recorded roles did not demonstrate eight independent analyses.
- A-D-I was recorded generically and must become hypothesis-specific.
- The 20 skills were loaded, but there is insufficient evidence to call them
  semantically applied.
- `save-run` path handling is a priority data-flow improvement case.
- `eval` and `JSON.parse` remained observations, not confirmed vulnerabilities.
- Python induction was undetermined because module loading failed.
- Historical comparison must include scope and agent/skill versions.
- Red-team and full suite validate Forge, not the security of `phylo`.
